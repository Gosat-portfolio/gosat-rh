Como enviar pelo Beefree

Siga os passos abaixo para enviar o e-mail formatado

1
Clique em Copiar HTML no painel ao lado para copiar o código do e-mail.
2
Acesse beefree.io e faça login (ou crie uma conta gratuita).
3
Clique em New Message → Import HTML e cole o código com Ctrl + V.
4
Clique em Send Test para testar ou em Export / Send para disparar o e-mail direto para o candidato.
5
Configure o remetente como rh@gosat.org nas configurações de envio do Beefree.
